# 🔍 Google Clone (Full Stack)

A modern Google Search Clone built using HTML, CSS, JavaScript, React, and Node.js. This project replicates the Google search interface and includes backend logic with a crawler and server for handling search functionality.

---

## 🚀 Features

- 🔍 Google-like search UI  
- ⚛️ React-based SearchBar component  
- 🌐 Backend server using Node.js  
- 🕷️ Custom crawler for data fetching  
- 🎨 Clean and responsive design  
- ⚡ Full-stack architecture  
- 💻 Lightweight and fast  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- React (SearchBar.jsx)  
- Node.js (server.js)

## ⚙️ How to Run

### 1️⃣ Clone the repository
git clone https://github.com/rsamwilson2323-cloud/google-clone-main.git

### 2️⃣ Navigate into the folder
cd google-clone-main

### 3️⃣ Install dependencies
npm install

### 4️⃣ Start the server
node server.js

### 5️⃣ Open the frontend
Open index.html in browser

---

## 🎯 Usage

1. Open the app  
2. Enter a search query  
3. Frontend sends request to backend  
4. Server processes data using crawler  
5. Results are displayed (based on implementation)
